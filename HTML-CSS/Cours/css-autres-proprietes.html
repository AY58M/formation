7) Les styles dynamiques

En CSS, on a ce qu'on appelle du style dynamique. Ce sont des propriétés qui vont s'appliquer sur un elément html lorsque qu'un action est effectué. Comme le passage de la souris, un clic, un double clic, un clic enfoncé, ...

Pour appliquer un style dynamique en css, on respecte la syntaxe suivante :

            sélecteur:nom_de_l_action {
                style css a appliquer
            }
    
            EXEMPLE :
    
            a:hover {
                font-style: bold;
            }
Passez la souris sur cet élément pour modifier le texte en gras, italique, et sa couleur
Maintenez le clic sur cet élément pour changer la couleur du texte

Il en existe beaucoup d'autres ...
